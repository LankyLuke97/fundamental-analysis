# System Design Document for Initial Portfolio Management System

## 1. Introduction
### 1.1 Purpose of the system

This document lays out a system to optimally manage a portfolio of shares and cash based on optimal portfolio theory. The aim of the system is, when provided with a set of company shares in which to make investments and a sum of available cash with which to do so, to provide the optimal weightings of capital allocation across the various options.
